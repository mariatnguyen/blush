# Blush

Blush is a mock eCommerce store built with [React](http://reactjs.org/).

Work in progress...

Features (will) include: 

* Filter per category
* Add to cart
* Add to favorites

![preview](https://raw.githubusercontent.com/mariatnguyen/blush/master/public/preview.png)

## Install

```bash
npm install
```

## Usage

```bash
npm start
```
